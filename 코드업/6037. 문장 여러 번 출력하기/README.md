# 문장 여러 번 출력하기

### 문제 설명

반복 횟수와 문장을 입력받아 여러 번 출력해보자.

### 예시

```py
n = input()
s = input()
print(int(n)*s)
```

### 참고

문자열 _ 정수 또는 정수 _ 문자열은 그 문자열을 여러 번 반복한 문자열을 만들어 준다.

### 입력

반복 횟수와 문장이 줄을 바꿔 입력된다.

### 출력

입력된 횟수만큼 입력된 문장을 출력한다.

### 입력 예시

3
I love CS

### 출력 예시

I love CSI love CSI love CS
